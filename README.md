# Accessible Contact Form (HTML Only)

This project is a **pure semantic HTML contact form** built with **web accessibility best practices**.  
It uses **no CSS, no JavaScript**, and relies entirely on **native HTML and ARIA attributes** to ensure the form is usable by screen readers, keyboard users, and assistive technologies.
